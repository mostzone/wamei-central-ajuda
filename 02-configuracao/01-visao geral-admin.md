# Visão Geral Admin

As documentações agrupadas nesta seção representam os primeiros e mais importantes passos para preparar sua conta Wamei Chat para o atendimento. Estas configurações são fundamentais para o funcionamento adequado da plataforma e devem ser realizadas antes de iniciar as operações de atendimento.

Esta área do sistema é destinada a administradores que terão a responsabilidade de configurar e manter a estrutura de atendimento da organização. Compreender cada uma destas configurações garantirá uma operação eficiente e organizada.

## Opções de Configuração Administrativa

O painel Admin oferece diversas opções de configuração que permitem personalizar a plataforma conforme as necessidades da sua operação. Abaixo, apresentamos uma visão geral de cada área:

### 1. Canais de Comunicação

O primeiro passo para qualquer operação de atendimento é conectar os canais por onde seus clientes entrarão em contato. Esta seção detalha como conectar cada tipo de canal suportado pela plataforma Wamei Chat.

**Tópicos incluídos nesta seção:**

- **Visão Geral dos Canais de Comunicação:** Apresentação dos tipos de canais disponíveis e suas características.
- **Conectando seus Canais (Passo a Passo):** Guia detalhado para realizar a conexão de cada tipo de canal.

A plataforma Wamei Chat oferece duas categorias de conexão para o WhatsApp: a API Oficial (WABA) e as APIs não oficiais (baseadas em leitura de QR Code). Além do WhatsApp, é possível conectar outros canais como Instagram, Facebook Messenger, Telegram, E-mail e Webchat.

### 2. Gestão de Atendentes

Com os canais conectados, o próximo passo é organizar a equipe que irá atender. Esta seção ensina como cadastrar os usuários (atendentes), agrupá-los em equipes e direcionar as conversas criando filas de atendimento.

**Tópicos incluídos nesta seção:**

- **Gestão de Usuários:** Como cadastrar membros da equipe na plataforma e definir seus respectivos perfis de permissão (Atendente, Supervisor, Admin).
- **Organização de Equipes:** A funcionalidade que permite agrupar usuários para fins de organização e distribuição de conversas.
- **Configuração de Filas de Atendimento:** As filas direcionam as conversas recebidas para usuários ou equipes específicas, como Vendas ou Suporte.

### 3. Horário de Atendimento

Defina o horário de funcionamento da sua empresa. Esta configuração é fundamental para o roteamento de conversas e para as mensagens automáticas de fora do expediente. O sistema oferece controle granular com horários distintos que podem ser definidos em três níveis: por Fila, por Usuário e por Tenant (Global).

### 4. Configurações Gerais Admin

Ajustes gerais da conta que afetam o comportamento do sistema, incluindo a ativação de bots, integrações com outras plataformas e configurações do CRM interno.

**Tópicos incluídos nesta seção:**

- **Configurações Gerais:** Configurações básicas do sistema.
- **Configuração de Bots e IA:** Ativação e configuração de chatbots e inteligência artificial.
- **Integrações:** Conexão com sistemas externos e APIs.
- **CRM:** Configuração do sistema de gestão de relacionamento com clientes.
- **Sessões:** Gerenciamento de sessões de atendimento.

## Ordem Recomendada de Configuração

Para uma configuração adequada da plataforma, siga a ordem recomendada abaixo:

1. **Primeiro:** Configure os Canais de Comunicação para definir por onde os clientes poderão entrar em contato.

2. **Segundo:** Configure os Horários de Atendimento para definir a disponibilidade da empresa.

3. **Terceiro:** Realize a Gestão de Atendentes, criando usuários e organizando equipes.

4. **Quarto:** Configure as Filas de Atendimento para direcionar as conversas corretamente.

5. **Quinto:** Ajuste as Configurações Gerais conforme as necessidades específicas da sua operação.

Esta ordem garante que você tenha a estrutura básica pronta antes de começar a receber atendimentos, evitando problemas de direcionamento e disponibilidade.
