# Kanban de Atendimento

O Kanban de Atendimento é uma ferramenta do Wamei Chat que permite o Controle Visual de Atendimentos de forma personalizável. Os kanbans são totalmente personalizáveis, permitindo a criação de quantos forem necessários para o funcionamento dos setores de atendimento.

**Disponibilidade:** Disponível para os perfis Administrador, Supervisor e Usuário.

## Tipos de Kanban no Wamei Chat

O Wamei Chat possui duas visualizações de Kanban distintas, cada uma com uma finalidade específica:

### Kanban de Atendimentos

Esta visualização utiliza **Etiquetas (Tags)** para organizar visualmente os *atendimentos em andamento*. É ideal para a gestão operacional da equipe de suporte, permitindo visualizar o status de cada atendimento em tempo real.

Os kanbans de atendimento podem ser organizados por:
- Status do atendimento (novo, em andamento, aguardando cliente, etc.)
- Tipo de solicitação
- Prioridade
- Canal de origem
- Qualquer outra classificação relevante para a operação

### Kanban de Oportunidades

Esta visualização utiliza as **Etapas do Funil** para organizar as *oportunidades de negócio*. É a ferramenta para a gestão de processos comerciais, permitindo acompanhar o progresso de cada oportunidade desde o primeiro contato até o fechamento.

Para mais detalhes sobre o Kanban de Oportunidades, consulte a documentação de [Funil de Oportunidades](/funil).

## Acesso

Para acessar o Kanban de Atendimento, clique no ícone de **Kanban** no menu lateral do seu painel. A interface será carregada com os kanbans disponíveis para sua operação.

## Aba Kanban

Nesta aba, você encontrará os kanbans criados ou um controle visual dos atendimentos com etiquetas definidas.

**Funcionalidades e Exibição:**

- **Visualização por categoria:** É possível visualizar os atendimentos por categoria, criando quantos kanbans forem necessários para organizar diferentes aspectos da operação.
- **Organização de tickets:** O Kanban auxilia na organização de tickets e potencializa o painel como um CRM completo.
- **Cards de atendimento:** Quando um Kanban é inserido, os contatos se organizam em cards, mostrando:
  - Foto de Perfil
  - Nome do Cliente
  - Número do WhatsApp
  - Número do Ticket de Atendimento
- **Ações disponíveis:** Para cada card, estão disponíveis ações como:
  - **Retomar o atendimento:** Voltar a atender aquele cliente.
  - **Retirar o kanban atribuído:** Remover a etiqueta kanban daquele atendimento.
- **Movimentação entre colunas:** Os contatos podem ser movidos entre as colunas do kanban conforme o andamento do atendimento, mantendo os dados sempre atualizados.
- **Salvamento:** É necessário clicar em *Save* para salvar as alterações realizadas.

## Aba Etiquetas

Nesta aba, serão exibidos os cards criados de acordo com as etiquetas definidas para cada atendimento.

**Funcionalidade:**

- **Visualização pura:** Serve apenas para visualização, mostrando os atendimentos organizados pelas etiquetas configuradas.
- **Sem salvamento automático:** **Não é possível salvar etiquetas** ao manipular os cards diretamente nesta aba.

## Criação e Configuração de Kanbans

Para criar novos kanbans personalizados:

1. Acesse as configurações de Kanban através do painel Admin.
2. Clique em adicionar novo kanban.
3. Defina:
   - **Nome:** Identificação do kanban.
   - **Colunas:** As etapas ou status que o kanban terá.
   - **Etiqueta associada:** Qual etiqueta do sistema está relacionada a cada coluna.
4. Salve as configurações.

Cada kanban pode ter sua própria estrutura de colunas e regras de negócio, permitindo adaptar a ferramenta às necessidades específicas de cada operação.

## Melhores Práticas

Para aprovechar ao máximo o Kanban de Atendimento:

1. **Crie kanbans específicos:** Para cada tipo de atendimento ou fluxo de trabalho.
2. **Mantenha atualizado:** Mova os cards conforme o andamento real dos atendimentos.
3. **Revise diariamente:** Faça回顾 diário do kanban para identificar gargalos.
4. **Defina responsabilidades:** Cada card deve ter um responsável claro.
5. **Use cores e标签:** Facilite a identificação visual de prioridades e status.
6. **Limite WIP:** Defina limites de trabalhos em progresso por coluna para evitar sobrecarga.

## Integração com Outras Funcionalidades

O Kanban de Atendimento integra-se com outras funcionalidades do sistema:

- **Etiquetas:** Cada coluna do kanban está associada a uma etiqueta do sistema.
- **Filtros:** Os filtros do sistema podem ser aplicados na visualização do kanban.
- **Relatórios:** Dados do kanban alimentam relatórios de produtividade.
- **Automação:** Fluxos podem mover tickets automaticamente entre colunas do kanban.
