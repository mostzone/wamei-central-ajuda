# Primeiro Acesso

Esta seção reúne os guias para novos clientes, cobrindo o processo desde o pós-compra até a conexão do primeiro canal de atendimento. O objetivo é proporcionar uma experiência fluida e intuitiva para que você possa começar a utilizar a plataforma Wamei Chat o mais rápido possível.

O primeiro acesso é uma etapa fundamental para garantir que todas as configurações iniciais estejam corretas e que a equipe esteja preparada para operar. Por isso, preparamos um passo a passo detalhado que vai desde o recebimento das credenciais até a realização do primeiro atendimento real.

É importante ressaltar que o Wamei Chat opera em dois níveis principais de administração: o Superadmin e o Admin. Cada um possui funções específicas e complementares, que serão apresentadas em detalhes ao longo desta documentação.

## Estrutura dos Painéis

A plataforma Wamei Chat possui dois níveis distintos de administração, cada um com responsabilidades específicas que garantem o pleno funcionamento da operação de atendimento.

O **Superadmin** funciona como o "Dono do Shopping Center", sendo responsável por gerenciar a estrutura geral, os lojistas (clientes) e a aparência do sistema como um todo. Este painel é utilizado para configurações de alto nível, como gestão da assinatura, customização da marca para revenda (White-Label), gestão de clientes tenants e configurações globais da plataforma.

O **Admin** funciona como o "Gerente da Loja", gerenciando as operações de uma conta específica. Este painel é utilizado para conectar canais de comunicação, gerenciar atendentes, criar chatbots e visualizar relatórios da operação.

## Próximos Passos

Para começar a utilizar o Wamei Chat, recomendamos seguir a ordem abaixo:

1. Leia atentamente as informações sobre **Onboarding** para entender o processo de integração
2. Acesse o sistema pela primeira vez seguindo as instruções de **Primeiro Acesso ao Sistema**
3. Configure um usuário Administrador para gerenciar a operação
4. Conecte seu primeiro canal de comunicação seguindo as instruções de **Conectando seu Primeiro Canal**
